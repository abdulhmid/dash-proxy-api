API Documentation

Base URL: http://localhost:8080/api/v1

Admin Endpoints

POST/auth/login
Login dengan username & password admin → return JWT token
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H 'Content-Type: application/json' \
  -d '{"username":"admin","password":"admin123"}'
GET/users Admin
Daftar semua users
POST/users Admin
Buat user baru
curl -X POST http://localhost:8080/api/v1/users \
  -H "Authorization: Bearer <token>" \
  -H 'Content-Type: application/json' \
  -d '{"username":"team-xxx","email":"team@x.com","password":"pass123"}'
POST/api-keys Admin
Buat API key untuk user (key hanya muncul sekali)
curl -X POST http://localhost:8080/api/v1/api-keys \
  -H "Authorization: Bearer <token>" \
  -H 'Content-Type: application/json' \
  -d '{"user_id":"<user-id>","name":"My Key"}'
DELETE/api-keys/{id} Admin
Nonaktifkan API key

API Source Management

GET/api-sources Admin
Daftar semua sumber data API
POST/api-sources Admin
Daftarkan sumber data baru (wajib: url)
curl -X POST http://localhost:8080/api/v1/api-sources \
  -H "Authorization: Bearer <token>" \
  -H 'Content-Type: application/json' \
  -d '{
    "base_url": "https://api.example.com/getLocation",
    "name": "my-api",
    "username": "api-user",
    "auth_headers": "{\"Authorization\":\"Bearer xxx\"}",
    "extra_params": "{\"param1\":\"value1\"}",
    "timeout_ms": 15000
  }'
PUT/api-sources/{id} Admin
Edit sumber data (field optional, hanya yg dikirim yg diubah)
DELETE/api-sources/{id} Admin
Hapus sumber data

Proxy Endpoints

POST/proxy/{source}/getLocation X-API-Key
Proxy ke sumber data. Parameter dikirim sebagai form-urlencoded
curl -X POST http://localhost:8080/api/v1/proxy/cp-cobra/getLocation \
  -H "X-API-Key: ask-xxxx..." \
  -d "msisdn=6281234567890"
